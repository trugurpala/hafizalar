# Code Of Conduct

Hafizalar is intended to help people ship better software with clearer evidence.

## Expected Behavior

- Be direct and respectful.
- Discuss technical risk with evidence.
- Keep feedback focused on the work.
- Avoid harassment, threats, personal attacks, and discriminatory conduct.

## Reporting

Use GitHub issues for normal project concerns.

For sensitive conduct or security concerns, avoid posting private details publicly. Open a minimal issue asking for a private channel or use GitHub security reporting when available.

Maintainers may remove disruptive content and limit participation when needed to keep the project useful.
